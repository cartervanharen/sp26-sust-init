# Trash Sorting Game – Progress Check Summary (March 12, 2026)

## Overview
The team presented a functional prototype of a campus-based trash sorting game using GameMaker. The game integrates the University of St. Thomas campus map and allows users to click buildings to complete trash sorting challenges.

---

## Core Game Concept
- Players navigate a campus map with selectable buildings.
- Each building contains a trash sorting mini-game.
- Sorting accuracy turns building indicators green; incorrect sorting contributes to red.
- Goal: clean the entire campus by correctly sorting trash.

---

## Gameplay Mechanics
- Drag-and-drop trash sorting into recycling, compost, or landfill.
- Progressive difficulty: new items unlock over time.
- Buildings start with varying “fullness” levels.
- Multiplayer concept: friends can help clean other buildings simultaneously.

---

## Key Design Decisions
- Use real campus imagery for immersion.
- Customize trash items based on building type:
  - ARC → concessions (e.g., popcorn containers)
  - ASC → dining items (takeout containers, napkins, food scraps)
  - Fry Hall → residential items (bottles, light bulbs, food scraps)
  - Classroom buildings (OEC, McNeely, Library) → paper, wrappers, cans
- Include both:
  - Generic items (paper, cardboard, wrappers, cans)
  - Location-specific items
- Use Saint Thomas branding:
  - Purple + green color scheme
  - Sustainable St. Thomas logo

---

## Feedback from Rachel
- Replace Aquinas Hall with OSF Library (higher student traffic).
- Consider adding Fry Hall (residence hall representation).
- Strong preference for building-specific trash items.
- Provide additional item assets (ARC, dining, etc.).
- Ensure accessibility (contrast, readability).

---

## User Experience Improvements
- Add instructions explaining green/red indicators on the map.
- Immediate feedback during gameplay:
  - Visual (red/green flash)
  - Optional sound (secondary to visuals)
- End-of-level review screen:
  - Show mistakes
  - Explain correct bin and reasoning
- Avoid disruptive pop-ups during gameplay.
